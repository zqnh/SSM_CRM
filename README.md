# SSM_CRM
基于SSM框架的客户管理系统

#### 1.项目介绍
客户管理系统，基于B/S架构，使用SSM框架来对客户信息进行CRUD操作
#### 2.运用技术
**后端技术:** Eclipse+Spring+Spring+Mybatis+Mysql<br/>
**前端技术:** HTML5+easyui+jsp
